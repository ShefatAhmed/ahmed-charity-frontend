# How to setup this website and how to use it

1. First you can clone this repo and this server repo(server repo link is bottom)
2. Then you can open it in terminal and type npm run dev both server and this frontend
3. You need to type both npm i -y
4. then you can add some information in server
5. after run the server you can get a local host link you can copy and replace it this frontend redux baseApi
6. I think the both application run successfully
7. This website made for react with typescript and redux state management, tailwind and some react libray etc.
